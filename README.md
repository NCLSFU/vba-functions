# VBA Functions

> A collection of reusable VBA functions — String, Date, Excel, and more.

## Structure

```
vba-functions/
├── String/          # String manipulation functions
├── Date/            # Date/Time functions
├── Excel/           # Excel操作 functions
├── Array/           # Array functions
└── Utils/           # General utilities
```

## Usage

Import `.bas` files into your Excel VBA project:

1. Open Excel → Alt + F11 (VBA Editor)
2. File → Import File → Select `.bas` file
3. Call functions from your modules
